Case Study: Drone Show Simulation

Motivation

Modern drone research often requires two things that don't mix well: high-level formation planning and low-level manual control. This project bridges that gap, creating a single platform to rapidly test formation algorithms while still letting the researcher manually override or fine-tune individual drones.

What I Did

  • Web Interface: Built a Flask-based web UI serving as mission control.
  • CV Pipeline: Developed OpenCV pipeline to map 2D images (like a triangle) into 3D GPS waypoints.
  • Simulation Backend: Integrated PX4 Autopilot SITL with Gazebo 3D simulation for realistic physics.
