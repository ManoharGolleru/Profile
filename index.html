Drone Swarm Simulation

The Challenge

Testing drone swarms in real life is expensive and dangerous. We needed a way to test formation algorithms safely.

The Solution

I built a pipeline connecting a Flask web interface to a Gazebo physics simulation. This allows researchers to upload an image, which is converted into GPS waypoints for the swarm to form in 3D space.
